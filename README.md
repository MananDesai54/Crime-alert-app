# How_To_Run

- Install Node.js and mongoDB

- ## How To run backend API

- Follow below [.env](#env) structure

```
$ git clone https://github.com/MananDesai54/Crime-alert-app.git
$ cd Crime-alert-app

$ npm install

$ npm start
```

- ## How To run Front-end App

- Install Node.js and expo-cli

```
$ git clone https://github.com/MananDesai54/Crime-alert-app.git
$ cd Crime-alert-app

$ cd app
$ npm install

$ npm start
```
